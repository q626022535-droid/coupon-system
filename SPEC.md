# 券码领取系统 - 需求文档

## 一、系统概述
- **系统名称**：券码领取系统
- **系统类型**：H5网页应用（微信内嵌）
- **技术栈**：Python + Streamlit

## 二、角色定义
| 角色 | 权限 |
|------|------|
| 超级管理员 | 最高权限，可管理所有审批人 |
| 审批人 | 审批用户申请、查看统计 |
| 普通用户 | 领取券码、查看申请记录 |

## 三、功能模块

### 3.1 券码管理（管理员）
- 上传券码文件（txt格式，每行一个券码）
- 设置券码金额（如：100元、200元）
- 查看券码库存状态（待领取/已发放/已使用）
- 提供上传模板下载

### 3.2 券码领取（用户）
- 浏览可领取的券码列表（显示金额）
- 选择券码后填写：事业部、姓名
- 提交申请，等待审批

### 3.3 审批流程（审批人）
- 查看待审批的申请列表
- 审批通过 → 券码发放给用户
- 审批拒绝 → 退回给用户
- 查看所有券码状态统计

### 3.4 审批人管理（超级管理员）
- 添加/删除审批人
- 分配审批权限

## 四、数据模型

### 4.1 券码表 (coupon)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| code | str | 券码 |
| amount | float | 金额 |
| status | str | pending/issued/used |
| issued_to | str | 领取人姓名 |
| department | str | 事业部 |
| apply_time | datetime | 申请时间 |
| issue_time | datetime | 发放时间 |

### 4.2 用户表 (user)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| username | str | 用户名 |
| password | str | 密码(加密) |
| role | str | admin/approver/user |
| department | str | 事业部 |
| name | str | 姓名 |

## 五、页面结构
1. 登录页
2. 用户首页（可领券码列表）
3. 申请记录页
4. 审批人首页（待审批列表）
5. 审批人审批页
6. 管理员首页（券码管理、上传）
7. 管理员审批人管理页
